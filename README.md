# Street Culture

> Heavyweight hoodies, raw graphic apparel, and limited-edition collaborative skateboard decks. Born in the underground, engineered for the concrete. Access member-exclusive drops via the Drop Club or locate your nearest flagship retail space in Seoul, NYC, and beyond.

Street Culture by Haru Commerce is a multi-tenant retail ecosystem fusing underground skate aesthetics with enterprise headless engineering. Part of the Haru modular brand architecture, alongside Haru Properties and Haru Agencies. All rights reserved.

## Features

### Site

- [ ] The site has internationalization, it can be viewed in both Korean and English Language, as well as USD and Korean Won
- [ ] The site has dark and light mode
- [ ] The site has smart recommendations like **You may also like** feature by checking out user’s wishlist
- [ ] The site has urgency system such as **only 3 left in stock** and **countdown timer** for discount
- [ ] The site has search functionality for products and blogs
- [ ] The site has Career, that people can apply jobs for opening positions
- [ ] The site has smart notifications for user’s membership expiration date
- [ ] The site has store pickup availability notification
- [ ] The site has store locator that can be searched with city name, and zip
- [ ] The site has Shop the look Feature
- [ ]

### Users

- [ ] User can subscribe to newsletter to receive up to date sales and information
- [ ] User can contact to the site directly with embedded contact form
- [ ] User can create account with email & password, oAuth
- [ ] User can edit personal information
  - [ ] update profile photo
  - [ ] update shipping & billing addresses
  - [ ] update basic info
  - [ ] update password
  - [ ] update email
  - [ ] can cancel or get monthly membership
- [ ] User can check the purchase history
- [ ] User can have personal Wishlist
- [ ] User can buy the product instantly with any payment method and checkout instantly
- [ ] User can review products
- [ ] User can subscribe to the monthly membership subscription **Drop club** to get member only discount & member only products & member only blogs
- [ ] User can save up points from every purchase that later using points to get exchangeable gifts
- [ ] User can leave comments on blogs
